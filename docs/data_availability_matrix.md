## Notes

1. **Season Format**:
   - NCAA/G-League/WNBA: `"2025"` (single year)
   - EuroLeague: `"E2024"` (E prefix)
   - LNB: `"2024-2025"` (4-digit format)

2. **Real-time vs Post-game**:
   - Real-time (15-min delay): NCAA, G-League, WNBA
   - Real-time: EuroLeague, EuroCup
   - Post-game: All others

3. **Scaffold leagues** (ACB, parts of NZ-NBL) require Selenium/Playwright for JS-rendered sites

4. **LNB sub-leagues** are the only international leagues with complete 7/7 dataset coverage AND name-based querying
